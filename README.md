## ğŸš€ Quick Start

### 1. Update Booking
Edit `index.html` and replace the Calendly embed URL with your actual Calendly link:
```html
<div class="calendly-inline-widget" data-url="https://calendly.com/YOUR_LINK" ...></div>
```

### 2. Add YouTube Video
Replace the video placeholder with actual embed:
```html
<iframe src="https://www.youtube.com/embed/YOUR_VIDEO_ID" ...></iframe>
```

### 3. Add Google Maps
In `contact.html`, replace map placeholder with:
```html
<iframe src="https://www.google.com/maps/embed?..." ...></iframe>
```
